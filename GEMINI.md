# Gemini CLI - Project Instructions

## Project Context
This is the `olxbot` project, a Telegram bot poller for OLX.in.

## Workflow Mandates
- **Sync Progress:** After completing a task, update `docs/PROGRESS.md`.
- **Verify Architecture:** Refer to `docs/CONTEXT.md` before making structural changes.
- **Maintain Standards:** Adhere to `docs/BEST_PRACTICES.md`.
- **Git Hygiene:** Work in the `develop` branch for active development; use `features/*` for new components.

## Subagent Guidance
- Use `codebase_investigator` to map existing logic before refactoring.
- Use `generalist` for batch updates to documentation or minor fixes.
